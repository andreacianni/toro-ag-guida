# DIVI Layout System

Il sistema di layout di DIVI utilizzato per il sito Toro AG è organizzato in una struttura gerarchica che permette di gestire l'aspetto visivo e la presentazione dei contenuti in modo flessibile e modulare.

## Panoramica del Sistema

Il sito utilizza diversi tipi di layout DIVI per gestire le diverse sezioni e tipi di contenuto. Ogni layout è progettato per uno scopo specifico e può essere personalizzato attraverso l'editor DIVI.

---

## Header & Footer

L'header e il footer sono elementi globali che appaiono su tutte le pagine del sito.

### Header (Intestazione)
L'header contiene:
- Logo aziendale
- Menu di navigazione principale
- Elementi di utilità (lingua, ricerca, etc.)

### Footer (Piè di pagina)
Il footer include:
- Informazioni aziendali
- Link utili
- Contatti
- Social media
- Copyright

**Dove modificarli**: Questi elementi vengono gestiti attraverso il Theme Builder di DIVI.

---

## Layout del Prodotto

Il layout utilizzato per visualizzare la pagina di dettaglio di un singolo prodotto.

### Struttura
Ogni pagina prodotto include:
- Immagine in evidenza
- Titolo del prodotto
- Descrizione completa
- Schede prodotto (PDF scaricabili)
- Documenti tecnici
- Video dimostrativi
- Galleria immagini (opzionale)
- Pulsante "Attivalo Ora" (opzionale)
- Prodotti correlati

### Utilizzo
Questo layout viene applicato automaticamente a tutti i post di tipo "Prodotto".

---

## Layout del Tipo di Prodotto

Il layout utilizzato per le pagine che raggruppano prodotti dello stesso tipo (categorie di prodotto).

### Struttura
Le pagine tipo prodotto includono:
- Immagine hero
- Titolo e descrizione del tipo
- Griglia dei prodotti appartenenti al tipo
- Schede tecniche del tipo
- Documenti e video correlati

### Utilizzo
Questo layout viene applicato alle taxonomy page dei tipi di prodotto.

---

## Layout dell'Applicazione

Il layout utilizzato per visualizzare le pagine delle applicazioni/colture.

### Struttura
Le pagine applicazione includono:
- Immagine rappresentativa
- Titolo dell'applicazione
- Descrizione dettagliata
- Brochure scaricabili
- Elenco dei prodotti consigliati per l'applicazione
- Contenuti multimediali correlati

### Utilizzo
Questo layout viene applicato alle taxonomy page delle applicazioni.

---

## Layout delle Pagine di Servizio

Layout utilizzati per pagine speciali come:
- Chi siamo
- Contatti
- Assistenza
- News/Blog
- Pagine istituzionali

### Caratteristiche
Questi layout sono più flessibili e possono essere personalizzati caso per caso attraverso l'editor DIVI per adattarsi alle esigenze specifiche di ogni pagina.

---

## Layout delle Pagine

Altri layout utilizzati per pagine generiche del sito.

### Tipologie
- **Homepage**: Layout speciale per la pagina principale
- **Archivio Prodotti**: Pagina indice di tutti i prodotti
- **Archivio Applicazioni**: Pagina indice di tutte le applicazioni
- **Pagine standard**: Template di base per contenuti generici
- **Pagine personalizzate**: Layout custom per esigenze specifiche

### Gestione
Questi layout vengono gestiti attraverso:
- Theme Builder di DIVI (per layout globali)
- Editor DIVI (per pagine specifiche)
- Library DIVI (per elementi riutilizzabili)

---

## Come Accedere ai Layout

Per modificare o visualizzare i layout DIVI:

1. Accedi alla Dashboard WordPress: `https://www.toro-ag.it/wp-admin/`
2. Vai su **Divi > Theme Builder**
3. Seleziona il layout che desideri modificare
4. Usa l'editor visuale per apportare modifiche

**Importante**: Le modifiche ai layout globali (Header, Footer, Template) avranno effetto su tutte le pagine che utilizzano quel layout.
